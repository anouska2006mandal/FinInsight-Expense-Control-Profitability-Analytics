# FinInsight: Expense Control & Profitability Analytics

## Project Overview
FinInsight is a financial analytics project developed using Microsoft Excel and MySQL to analyze budgets, expenses, revenue, and profitability. The project provides an interactive dashboard for monitoring financial performance and supports data-driven decision-making.

## Project Objectives
- Analyze financial datasets efficiently.
- Compare planned budgets with actual expenses.
- Calculate key financial KPIs.
- Perform variance and profitability analysis.
- Develop an interactive financial dashboard.
- Use SQL for financial data storage and analysis.

## Tools & Technologies
- Microsoft Excel
- MySQL
- phpMyAdmin
- XAMPP

## Project Deliverables
- Finance analytics dataset with metric definitions.
- Variance and profitability dashboards.
- Budget control recommendation report.

## Folder Contents
- FinInsight_Project_Report.pdf – Complete project report.
- FinInsight_Presentation.pptx – Project presentation.
- FinInsight_Dashboard.xlsx – Excel dashboard and analysis.
- SQL_Queries.sql – SQL queries used for analysis (if included).

## Key Features
- KPI Dashboard
- Budget vs Actual Expense Analysis
- Variance Analysis with Alerts
- Product Profitability Analysis
- Executive Dashboard
- SQL Database Analysis

## How to Use
1. Open the Excel workbook to view the dashboard and financial analysis.
2. Open the project report for detailed documentation.
3. Review the PowerPoint presentation for the project summary.
4. Execute the SQL queries using MySQL/phpMyAdmin if required.
